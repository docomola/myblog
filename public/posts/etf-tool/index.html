ETF 投资工具开发笔记

最近在开发一个 ETF 排序工具，帮助快速筛选溢价率合理的 ETF 产品。

功能特性

  • 📊 实时排序（按溢价率、成交量、规模）
  • 🔍 关键词搜索
  • 📈 溢价率实时显示（修复了显示为0的 bug）
  • 💾 数据导出

技术实现

使用纯前端实现，数据来源于公开 API：

async function fetchETFData() {
  const res = await fetch(API_URL);
  const data = await res.json();
  return data.filter(etf => etf.premiumRate !== 0);
}
