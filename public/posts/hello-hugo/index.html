Hugo 博客 教程

欢迎来到我的 Hugo 博客！

这是使用 Hugo + Mana 主题 搭建的个人博客的第一篇文章。

为什么选择 Hugo？

Hugo 是一个非常快速的静态站点生成器，具有以下优点：

  • ⚡ 极速构建 — 数千篇文章也能在毫秒级完成构建
  • 🛠️ 无需依赖 — 单个二进制文件，部署简单
  • 📝 内容优先 — 使用 Markdown 写作，专注内容创作
  • 🌍 丰富的主题生态 — 社区提供了大量免费主题

Mana 主题特点

Mana 是一个简洁优雅的 Hugo 博客主题：

  1. 响应式设计，移动端友好
  2. 支持深色模式
  3. 内置文章搜索功能
  4. 支持代码高亮（Catppuccin 主题）
  5. 简洁的卡片式文章列表

示例代码

def hello_hugo():
    print("Hello, Hugo!")
    print("欢迎来到我的博客！")

hello_hugo()
